## 2. Comandos de Git utilizados

- `git clone`  
  Se usa para copiar un repositorio.

- `git status`  
  Muestra el estado de los archivos del repositorio.

- `git add .`  
  Agrega todos los archivos al área de preparación para el commit.

- `git push`  
  Sube los cambios al repositorio en GitHub.

- `git branch`  
  Muestra o crea ramas dentro del repositorio.

- `git checkout`  
  Permite cambiar entre ramas o versiones del proyecto.
